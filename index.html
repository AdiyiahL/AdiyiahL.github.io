What is SpaCy?

SpaCy is an open-source natural language processing (NLP) library designed to be efficient, scalable, and production-ready. It provides a simple and intuitive API for processing and analyzing textual data, making it a popular choice among developers and researchers working with NLP tasks.

Key Features of SpaCy

Tokenization
Efficient tokenization and linguistic annotations. Segmenting text into words, punctuations marks etc.
Part-of-speech tagging
Assigning word types to tokens, like verb or noun.
Named entity recognition
Labelling named “real-world” objects, like persons, companies or locations.
Syntax parsing
Assigning syntactic dependency labels, describing the relations between individual tokens, like subject or object.
Similarity
Word vectors and similarity. Comparing words, text spans and documents and how similar they are to each other.
Text classification
Assigning categories or labels to a whole document, or parts of a document.
Custom rule-based matching.
Finding sequences of tokens based on their texts and linguistic annotations, similar to regular expressions.
Training
Training of machine learning models. Updating and improving a statistical model’s predictions.
Integration
Easy integration with other libraries and frameworks

How to use SpaCy?

Installation

To install SpaCy, you can use pip, the Python package manager, by running the following command:

pip install spacy

Tokenization

Import the spacy package, call the load method, store the English model to be applied in the variable nlp, and use the model to analyze the text. 

              
              text = 'My daughter, Jane, never dreamed of receiving a letter
              from a girl of her own age in Holland. Last year, we were travelling
              across the Channel and Jane put a piece of paper with her name and address on it into a bottle.
              She threw the bottle into the sea. She never thought of it again,
              but ten months later, she received a letter from a girl in Holland.
              Both girls write to each other regularly now.
              However, they have decided to use the post office.
              Letters will cost a little more, but they will certainly travel faster.'
              
              import spacy
              nlp = spacy.load('en')
              doc = nlp(text)

We try to use Spacy to analyze all tokens that appear in the text.

              for token in doc:
                 print('"'+token.text+'"')

The output shows some attributes of the first 10 tokens: text, index value, token, whether it is a punctuation mark, whether it is a space, and a mark; 

              
              for token in doc[:10]:
                  print("{0}\t{1}\t{2}\t{3}\t{4}\t{5}\t[6]".format(
                  token.text,
                  token.idx,
                  token.lemma_,
                  token.is_punct,
                  token.is_space,
                  token.tag_

Named entity recognition

First, we analyze the entity vocabulary (entity) that appears in the full text;

                
                for ent in doc.ents:
                    print(ent.text, ent.label_)

As can be seen from the figure below, Spacy has extracted all entities such as names, places, and dates in this text:

Secondly, calling the dislacy method can visualize the entity vocabulary;

                
                from spacy import displacy
                displacy.render(doc, style='ent', jupyter=True)

Syntax Parsing

Next, we analyze and visualize the dependency relationship of word cases in a sentence: First of all, if you want to analyze a single sentence, you need to convert it into a list; 

                
                list(doc.sents)

Then extract the first sentence and process it with the nlp model, and store it in a new variable:

                
                newdoc = nlp(list(doc.sents)[0].text)

Second, we use a for loop to output the dependencies between tokens; 

                
                for token in newdoc:
                    print("{0}/{1} <--{2}-- {3}/{4}".format(
                        token.text, token.tag_, token.dep_, token.head.text, token.head.tag_))

Then call the display method for visualization: 

                
                displacy.render(newdoc, style='dep', jupyter=True, options={'distance':60})

A Simple Example

After downloading the required language model, you can start using SpaCy in your Python scripts. Here's a simple example that demonstrates tokenization and part-of-speech tagging:

import spacy # Load the English language model nlp = spacy.load('en') # Process a text string doc = nlp("SpaCy is an amazing tool for NLP.") # Accessing tokens and their part-of-speech tags for token in doc: print(token.text, token.pos_)
